🌱 Plant Pest Identifier and Treatment Recommender

An AI-powered system that identifies agricultural pests from images and recommends suitable treatment methods to help farmers protect crops and improve productivity.








📖 Overview

Agricultural pests cause significant crop damage and economic loss worldwide. Early and accurate pest identification is essential for effective pest management.

This project uses Deep Learning and Computer Vision techniques to automatically identify pests from images and provide treatment recommendations to farmers, agricultural professionals, and researchers.

The system aims to reduce manual inspection efforts and support smart agriculture practices.

🎯 Objectives
Detect and classify plant pests from images.
Improve pest identification accuracy using deep learning.
Recommend appropriate treatment methods.
Support sustainable and smart agriculture.
Provide a user-friendly interface for pest diagnosis.

📂 Dataset

This project utilizes the IP02 / IP102 agricultural pest dataset, which contains images of various insect pest species used for training and evaluation. The dataset provides a large-scale benchmark for pest recognition tasks in agricultural environments.
